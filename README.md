# Extract_urls
一个脚本:\<br />
*****************************************\<br />
*功能: \<br />
*1.提取参数,如:pytho<br /> Extract_urls.py -h中的h进行处理\<br />                 
*2.打开保存url的文件,with...as...方式\<br />
*3.从url中提取首页,正则表达式\<br />
*4.将首页地址保存到数据库(MySql/Mo<br />goDB)\<br />
*5.geve<br />t+requests异步下载首页内容html\<br />
*6.爬取页面图片作为附件\<br />
*7.将附件和html通过smtp邮件发送\<br />
*****************************************\<br />
